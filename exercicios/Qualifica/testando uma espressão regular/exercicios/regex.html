Exercicio 03

    Crie uma regex que só aceite strings terminadas com ID;
  • const validarId = /\d+ID\b/;
    console.log(validarId.test("3456582ID"));
    console.log(validarId.test("3456582"));
    console.log(validarId.test("asd"));
    console.log(validarId.test("asdasdID"));
    console.log(validarId.test("3456582ID"));

Exercici0 04

    Crie uma regex que aceite a seguinte expressão "Marca: nomeDaMarca";
    Onde nomeDaMarca pode variar para Nike, Adidas, Puma, Asics;
    Depois teste;
  • const validarMarca = /Marca: (Nike|Adidas|Puma|Asics)/; console.log(validarMarca.test("Marca: Nike"));
    console.log(validarMarca.test("Marca: asd"));
    console.log(validarMarca.test("Marca:"));
    console.log(validarMarca.test("123321"));
